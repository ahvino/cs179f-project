Embedded Systems Projects

Some of the projects I've worked on during my time @ UCR

By Selik Samai

This is a very basic game I made. It was from my first embedded systems course and the game is called wall smasher. Basically it's a timing game where the idea is to punch through the wall before it hits your character. The game has a point system and a given number of lives and walls that the player needs to clear. Here is a list of peripherals and componenets I used in the project.

  • 7-seg display
  • LCD screen
  • Speaker
  • shift registers
  • 3 buttons
  • led-bar

The project was only about a week works of work and given that it was an intro course to Embedded Systems, it made sense. Source code can be found here
